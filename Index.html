Protocolos consultados

HTTP (Protocolo de transferencia de hipertexto):

HTTP es el protocolo utilizado para transferir datos en la World Wide Web. Funciona sobre el protocolo de transporte TCP/IP y se utiliza para solicitar y enviar páginas web, imágenes, vídeos y otros recursos en línea. HTTP define cómo se deben formatear y transmitir las solicitudes y respuestas entre un cliente (como un navegador web) y un servidor web.

TCP/IP (Protocolo de control de transmisión/Protocolo de Internet):

TCP/IP es un conjunto de protocolos que se utilizan para conectar dispositivos en redes, incluido Internet. El Protocolo de Control de Transmisión (TCP) maneja la entrega de datos de manera confiable y ordenada, mientras que el Protocolo de Internet (IP) maneja el direccionamiento y enrutamiento de los datos a través de la red.

DNS (Domain Name System):

DNS es un sistema que traduce los nombres de dominio legibles para los humanos (como www.portafolio.com ) en direcciones IP numéricas que las computadoras utilizan para identificarse entre sí en Internet. Esto permite a las personas acceder a sitios web utilizando nombres de dominio en lugar de recordar direcciones IP.

CDN (Content Delivery Network):

CDN (Content Delivery Network): Una CDN es una red de servidores distribuidos geográficamente que trabajan juntos para entregar contenido web de manera eficiente. Al almacenar copias de contenido (como imágenes, videos y archivos) en varios servidores en diferentes ubicaciones, las CDN reducen la latencia y mejoran los tiempos de carga al servir contenido desde un servidor cercano al usuario.

WWW (World Wide Web):

La World Wide Web es un sistema de información en línea que permite a los usuarios acceder y navegar a través de documentos multimedia, enlaces y recursos en forma de páginas web. Fue desarrollado por Tim Berners-Lee en la década de 1980 y se basa en tecnologías como HTTP, URL y HTML.

FTP (Protocolo de transferencia de archivos):

FTP es un protocolo utilizado para transferir archivos entre una computadora y un servidor a través de una red. Permite cargar, descargar y gestionar archivos en un servidor remoto. Aunque menos común en comparación con métodos más seguros como SFTP (FTP seguro) o SCP (Protocolo de Copia Segura), FTP todavía se utiliza en algunas aplicaciones.

HTML (Lenguaje de marcado de hipertexto):

HTML es el lenguaje de marcado utilizado para crear documentos web. Utilice etiquetas para estructurar y presentar contenido, como texto, imágenes, enlaces y otros elementos multimedia, en una página web. Los navegadores interpretan el HTML para mostrar el contenido web de manera visual.
